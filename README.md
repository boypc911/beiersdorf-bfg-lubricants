## Publish to GitHub Pages

1. Create a GitHub repository, for example `beiersdorf-bfg-lubricants`.
2. Upload all files in this folder: `index.html`, `styles.css`, `script.js`, and `assets/`.
3. Go to `Settings` -> `Pages`.
4. Choose `Deploy from a branch`.
5. Select `main` and `/root`, then save.
